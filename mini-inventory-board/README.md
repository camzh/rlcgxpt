# 货源看板小程序 MVP

## 项目说明

这是一个面向内部员工使用的微信小程序 MVP，用于统一录入、查看和更新服务器货源状态。

当前版本包含：

- 员工首次注册
- 管理员审核员工申请
- 货源看板首页
- 新增/编辑货源
- 货源详情
- 状态更新：跟进中、已售出、已下架
- 逻辑删除
- 操作日志时间线
- 我的页面
- 电视大屏展示数据预留接口

## 数据说明

当前使用微信本地存储模拟后端数据：

- `services/inventory.js`
- `services/auth.js`
- `mock/data.js`

首次打开会自动写入示例数据。

## 大屏预留

当前已预留 `getScreenPayload()`，用于未来电视端读取展示型数据。

该方法会输出：

- 汇总统计
- 可展示货源列表
- 展示优先级和展示标签

## 打开方式

1. 打开微信开发者工具
2. 导入目录 `C:\Windows\User\mini-inventory-board`
3. 使用“测试号 / 游客模式”打开
4. 编译即可预览
